# Cat Merge

A mobile puzzle game where players merge cats to create higher-level cats.

## Features
- Drag-and-drop interaction using Unity physics and colliders
- Merge mechanic that spawns higher-level cats when identical units collide
- Modular scripts designed with object-oriented programming principles
- Android platform

## Technologies
- Unity, C#
- OOP principles

## How to Play
Drag cats into each other to merge them. Merge identical cats to unlock higher-level cats and progress through levels.
